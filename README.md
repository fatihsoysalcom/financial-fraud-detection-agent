# Financial Fraud Detection Agent

This example demonstrates a simple agent-based system for financial fraud detection, a common application in BFSI. The `FinancialAgent` class processes transactions, applies predefined rules to identify suspicious activity, and makes autonomous decisions to flag or approve transactions. It illustrates how agents can automate decision-making and risk assessment in financial services.

## Language

`python`

## How to Run

1. Ensure you have Python 3 installed.
2. Save the code as `main.py`.
3. Run from your terminal: `python main.py`
